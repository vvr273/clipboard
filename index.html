How it works

  1. Paste text and click Send.
  2. A short code is generated and stored online.
  3. On the second device, open the same page and click Receive.
  4. Enter the code, retrieve the text, and copy it.

Free setup

This UI is static. For real cross-device sharing, connect it to Firebase on the free Spark plan and deploy on GitHub Pages, Netlify, or Vercel.
